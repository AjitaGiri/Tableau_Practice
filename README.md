# Sales Dashboard 
 
## Overview
This project is a **Tableau Sales Dashboard** built using sales data from multiple tables, including Sales, Product, Customer, Territory, Calendar, DimProductCategory, and DimProductSubcategory. The dashboard provides a **high-level overview of key business metrics** and allows interactive exploration using filters.

## Key Features

### KPIs
The dashboard includes **3 Key Performance Indicators (KPIs)** at the top:  
1. **Total Sales** – Total revenue generated from all sales.  
2. **Total Profit** – Calculated as `Sales Amount - Total Product Cost`.  
3. **Total Quantity** – Total number of products sold.

### Charts / Graphs
The dashboard includes **3 main visualizations**:  
1. **Sales by Product Category** – Bar chart showing revenue contribution of each product category.  
2. **Profit Trend Over Time** – Line chart showing monthly/quarterly profit trend.  
3. **Quantity Sold by Product Category** – Bar chart showing the number of products sold per category.
